Understanding the Models

Two different approaches to the same challenge‚Äîfinding patterns in price data.

üìè Linear Regression

Linear Regression is a foundational statistical method that fits a straight line through your data. It looks for a simple relationship between input features (like past prices or trading volume) and the target (future price).

  • Easy to interpret‚Äîyou can see exactly how each feature affects predictions
  • Fast to train and requires minimal computational resources
  • Works well when relationships in data are roughly linear
  • Struggles with complex, non-linear patterns in volatile markets
  • Best suited for identifying general trends rather than precise daily predictions

üß† LSTM (Long Short-Term Memory)

LSTM is a type of recurrent neural network specifically designed for sequential data. It can "remember" patterns across time, making it popular for time series like stock prices where past values influence future ones.

  • Captures temporal dependencies that simpler models miss
  • Can learn complex, non-linear relationships automatically
  • Requires more data and computational power to train effectively
  • Harder to interpret‚Äîunderstanding why it makes certain predictions is difficult
  • Prone to overfitting if not carefully tuned with proper validation

The Prediction Pipeline

From raw data to visual insights‚Äîhere's how StockVision processes information.

1

4

Trained models receive recent data as input and output predicted prices for the specified forecast period.
